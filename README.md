# 振荡器调音 DEMO

## 项目简介

支持声音频率修改、波形选择、音量调节和输出声音功能，可以支持多个声音频率和波形组合输出声音。

## Demo 示例

![DEMO](https://github.com/pengxiaohua/audio-oscillator-flow/blob/main/src/assets/demo.jpg)

## 安装流程

1. Clone

   ```bash
     git clone https://github.com/pengxiaohua/audio-oscillator-flow
   ```

2. Install

   ```bash
     npm install
   ```

3. Run

   ```bash
     npm run dev
   ```
